2
Go to Email Templates → Create template with these variables:
To: {{to_email}} | Subject: {{subject}}
Body: Dear {{to_name}}, {{message}} → Copy the Template ID
3
Go to Account → API Keys → Copy your Public Key
4
Paste all three below and click Save & Connect
